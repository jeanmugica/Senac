Objetivo.
O entity framework agiliza todo o processo de criação do banco de dados, diminui o codigo e otimiza 
o tempo do desenvolvedor.

Requisitos.
Visual Studio 2013 instalado, sql server.

Documentação.
Próprio site do entity framework ( www.asp.net/entity-framework ) e tambem o msdn.microsoft.com tem várias dicas, 
tudo em inglês.

Principais vantagens.
Foca no desenvolvimento da aplicação mais como um modelo ( Objetos ), agiliza o processo de criação do banco de dados,
facilitando a vida do programador.

Desvantagens.
Se não for bem planejado o framework acaba atrapalhando a vida do desenvolvedor.
